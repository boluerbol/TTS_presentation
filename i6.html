Dataset & Preprocessing

Custom Curated Kyrgyz Speech Corpus for KyrgyzTTS

Language Resource

Kyrgyz Language

High-quality recordings of native speakers, curated specifically for TTS synthesis.

Audio Specifications

22.05 kHz Mono

16-bit PCM WAV format. Sampled to balance high fidelity with efficient processing speed.

Data Format

LJSpeech Style

Standardized structure: wavs/ directory and filelist.txt containing path|transcript.

Acoustic Statistics

Mel-Mean & Std

Calculated via matcha-data-stats. Crucial for calibrating the model to the specific volume/pitch range.

Feature Extraction

80 Mel Bands

STFT parameters: FFT size 1024, Hop size 256. Optimized for HiFi-GAN vocoder compatibility.

Text Processing

Cyrillic Cleaners

Custom regex rules to normalize punctuation, expand numbers, and standardize Kyrgyz orthography.
